# HCC004v1 - Dysrhythmia Suspect

### Target Population
All patients with any active medication in the Antiarrhythmics drug class on their Medication List.

### Recommendations
When a patient has an active antiarrhythmic medication but no corresponding active condition in the Dysrhythmia class (ICD-10 I42.*, I47.*, I48.*, I49.*), the protocol surfaces a "Diagnose" recommendation. It prompts the clinician to consider updating the Conditions List to include a dysrhythmia-related problem as clinically appropriate. When a corresponding active dysrhythmia condition is already on file, the protocol card is marked satisfied. Patients without an active antiarrhythmic medication are not in the denominator and the protocol emits no effects.

### Importance
Patients on antiarrhythmic therapy almost always have an underlying dysrhythmia, and missing the diagnosis on the problem list weakens clinical documentation, risk adjustment, and downstream care coordination. Surfacing the gap at the point of charting helps keep the problem list aligned with the actively managed medications.

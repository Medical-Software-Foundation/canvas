# CCP004v1 — Diagnosis Of Diabetes

### Target Population
All patients in the practice.

### Recommendations
For any patient with an active diabetes condition (per the standard Diabetes value set), the protocol surfaces a "Contact the patient" recommendation on a protocol card and includes the date of the earliest active diagnosis in the narrative.

When a patient has no active diabetes condition, the protocol card is marked satisfied.

### Importance
This protocol provides a continuous, patient-level signal that a diabetes diagnosis is present and acknowledged in the chart, making it easy to drive downstream outreach or care-coordination workflows.

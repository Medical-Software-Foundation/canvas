# CCP003v1 — Diagnosis Of Hypertension

### Target Population
All patients in the practice.

### Recommendations
For any patient with an active hypertension condition (per the standard Diagnosis of Hypertension value set), the protocol surfaces a "Contact the patient" recommendation on a protocol card and includes the date of the earliest active diagnosis in the narrative.

When a patient has no active hypertension condition, the protocol card is marked satisfied.

### Importance
This protocol provides a continuous, patient-level signal that a hypertension diagnosis is present and acknowledged in the chart, making it easy to drive downstream outreach or care-coordination workflows.

EHI Export

Export complete electronic health information per patient.

Export a patient's complete electronic health information (EHI) — every FHIR resource Canvas holds for them — as one NDJSON file per patient.

Search or filter the patient list, then export a single patient with the Export button, or tick several and use Export selected — or Export all matching to export everyone in the current filter.

💡 No matter how many patients you choose, exports are queued and run automatically in small batches in the background, so your Canvas instance is never overloaded. You don't need to keep this page open — come back any time to check progress and download finished files under Export runs or each patient's Last export.

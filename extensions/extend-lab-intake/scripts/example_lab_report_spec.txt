Lab Report Extraction Specification
====================================

This processor should extract data from laboratory test result reports.

Patient Information:
- Patient's full name (first and last)
- Date of birth
- Medical record number (MRN) if present
- Phone number if present

Lab Information:
- Name of the laboratory
- Date the sample was collected
- Date the report was generated
- Ordering physician name

Test Results (can be multiple):
For each test result, extract:
- Test name (e.g., "Glucose", "Hemoglobin A1c", "TSH")
- LOINC code if available
- Result value (numeric)
- Unit of measurement (e.g., "mg/dL", "%", "mIU/L")
- Reference range (e.g., "70-100", "<5.7")
- Flag indicating if the result is abnormal (high, low, or critical)

Additional Notes:
- Any comments or notes from the lab
- Specimen type (blood, urine, etc.)

=================
